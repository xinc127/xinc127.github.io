Ubuntu 18.04 XMind安装及遇到的问题

发表于 2019-02-15 | 分类于 ubuntu软件
本文字数： 679 | 阅读时长 ≈ 1 分钟

XMind下载ubuntu压缩包

sudo unzip xmind.zip -d /opt/XMind

将压缩包解压至opt文件夹下的XMind，一般将软件安装在这里

创建快捷方式

sudo gedit /usr/share/applications/xmind.desktop

创建快捷方式文件夹

同样的，sudo chmod 777 xmind.desktop给文件权限；

另外许找一张128*128的图标文件放置/opt/XMind目录下；

遇到的问题

  • 提示文件读写权限问题

解决：

sudo chmod 777 XMind -R

解决后面打开XMind遇到读写权限问题

解决：

sudo gedit /opt/XMind/XMind_amd64/XMind.ini

文件添加下面代码

# XMind # ubuntu
