Vue CLI3路由懒加载无效

发表于 2019-02-25
本文字数： 859 | 阅读时长 ≈ 1 分钟

遇到的问题

Vue CLI3创建的项目，按照文档写法写的路由懒加载

const Foo = () => import('./Foo.vue')

首页加载时候还是会加载全部组件.js文件

解决

Vue CLI3文档HTML和静态资源部分对Prefetch部分介绍

<link rel="prefetch">是一种resource hint， 用来告诉浏览器页面加载完成后，利用空闲时间提前获取用户未来可能会访问的内容。

通过chainWebpack的config.plugin('prefetch')进行修改和删除

参考链接

Resource Hint

是一系列相关标准，来告诉浏览器哪些源下的资源我们的客户端想要获取，哪些资源在之后的操作或浏览时需要被使用，以便浏览器能够进行一些预先连接或预先加载等操作。

Resource Hint 有 DNS Prefetch 、Preconnect 、Prefetch 和 Prerender这四种。
