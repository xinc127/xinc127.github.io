创建新的文章

根目录下命令行输入hexo new <模板> <文章名> 新建文章

其中的模板在scaffolds文件夹中，默认有三种模板post、draft、page

参数 功能 路径
post 新建文章 /source/_posts
draft 新建草稿 /source/_draft
page 新建页面（标签页、分类页等） /source/

通常使用hexo new post <title>新建文章，这里的<titel>不是文章名字。

例如新建一篇文章 hexo new post ariticles

然后在source/_posts文件夹下面多了一个articles.md文件

打开文件后（使用的Typora编辑器打开）

在其中配置文章相关内容，标签、分类等。其中title为文章发表后的文章页面显示的标题。

而.md文件的名字，将作为文章的链接地址显示，例如本篇的文件名：articles.md，那么本篇文章的地址为http://xxx.xxx/ariticles.html

文章模板

在新建文章的post参数就是模板的名称，在使用hexo new post <title>命令新建文章时，系统会自动在scaffolds文件夹下寻找post文件，根据该配置文件生新的.md文件
